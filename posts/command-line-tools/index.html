Command Line Tools

October 11, 2020

Installing Command Line Tools

  • Open terminal, enter xcode-select --install to install the command line tools to your Mac.
  • You can use xcode-select --version to check the version installed.
  • To see the path where the command line tools is installed you can use this command: xcode-select --print-path.

Uninstall Command Line Tools

  • Open Finder
  • Select your drive, normally it is called Macintosh HD by default.
  • Go to Library folder and find a folder named Developer.
  • Inside Developer there will be a folder called Command Line Tools, simple delete the entire folder to delete the command line tools.
