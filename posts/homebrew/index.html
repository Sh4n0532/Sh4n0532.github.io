What is Homebrew

Homebrew is a package manager for MacOS and Linux. It helps us to install things you need in an easier way. Everything installed using Homebrew will be in its own directory so it is easier to manage them. It also provide many useful command to help you manage your packages.

How to Install?

To install Homebrew on Mac you need to install the command line tools first. Please visit Command Line Tools to see the step for installing it.

After that you can go to Homebrew homepage https://brew.sh and copy the link under “Install Homebrew” and paste it into your terminal to install Homebrew. Follow the instruction given.

To verify your installation, enter brew --version to see the version installed.

You can find Homebrew in path /usr/local/Homebrew/bin/brew. The packages installed using Homebrew can be find in /usr/local/Caskroom or /usr/local/Cellar depends on the type of package installed. Most GUI application can be found in Caskroom and CLI application in Cellar.

Homebrew Command

  • brew help or brew help <command>: to get help about a command and know how to use the command to perform operations.
  • brew search or brew search <keyword>: without specifying the package name it will list out all the available packages that you can install using Homebrew. With package name it will return only packages related to the keyword you searched. The result will be separated into Formula and Cask. Formulae contains package that operate inside command line while Cask is an extension of Homebrew that allow us to install MacOS native application and also other GUI apps such as Google Chrome.
  • brew install <package_name>: to initialize installation
  • which <package_name>: to show the path where the package is store but this is not the exact path, this is just the symlink. To show the exact path, enter ls -al <symlink>.
  • brew info <package_name>: to show more details of the package.
  • brew uninstall <package_name>: to remove the package.
  • brew list: list out the packages you have installed.
  • brew update: update packages and Homebrew.
  • brew outdated: show packages with update available.
  • brew upgrade: update all the outdated packages.
  • brew pin <package_name>: to prevent the package from being updated.
  • brew unpin <package_name>: allow the package to be updated.
  • brew cleanup or brew cleanup <package_name>: to remove older version of packages. This should be done frequently to free up some space.
  • brew cleanup -n: list all the packages that will be remove with brew cleanup but not actually removing it, dry run.
