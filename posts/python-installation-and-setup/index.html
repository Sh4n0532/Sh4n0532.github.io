Python Installation and Setup

October 12, 2020

Homebrew Method

To install Python using Homebrew you must have Homebrew available in your machine. Please go to this page to see the installation process of Homebrew.

Installing Python

  • brew install python to start the installation.
  • After the installation is done it will print out something like this: It provide you the path that you will need to add PATH if you want to use this version of Python as the default Python. This is because MacOS actually comes with Python 2.7 pre-installed and that will be your default Python if you did not configure the PATH.
  • Add the path provided export PATH="usr/local/opt/python@3.8/libexe/bin:$PATH to your .bash_profile or .zshrc file depending on the terminal you used. This will concatenate the Python path to your original PATH.
  • Restart the terminal to let the changes take place.
  • Now, you can check Python version and also pip version to verify your installation and making sure that it is your default Python now by entering python --version and pip --version.

Method Without Homebrew

  • Go to Python official website https://www.python.org
  • Click on the Download section. If you want the latest version of Python you can simply click the big yellow download button to start download, but if you want any specific version of Python for any reason you can click on your operation system below the download button to view all releases. It is recommended to go for the stable release.
  • After that just run the installer and follow the instruction to complete your installation. Then you should also check your version with python3 --version.
