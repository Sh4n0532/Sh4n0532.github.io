Java Installation and Setup

October 10, 2020

Install

You can install Java JDK through Oracle official website https://www.oracle.com/java/technologies/javase-downloads.html. Click on JDK Download and choose your operating system to start downloading. After that you just need to follow the normal installation process through the wizard.

After install Java for MacOS you need to set the JAVA_HOME path. Add this export JAVA_HOME="/usr/libexec/java_home -v 14" to your .bash_profile or .zshrc depending on your terminal type. Then you can use java --version to verify java version.

Install with Homebrew

Installing Java using Homebrew is a very easy process. But before that you need to make sure that you have Homebrew installed. If you don’t know what is Homebrew or how to install it please visit Homebrew

Before install Java, if you want more information about it there are fews command that you can use:

  • brew search java return the details of Java package and to make sure it is available
  • brew info java will show a very details information about Java such as the website it will install from, the version, dependencies and others.
  • To install Java enter brew install java to start the installation.
  • When done installed it will print some message that tell you what and how to configure to make Java working, follow the necessary steps depends on your needs. For the basic you should at least make Java working in your terminal. To make it work in terminal just add export PATH="/usr/local/opt/openjdk/bin:$PATH" to your .bash_profile or .zshrc file depends on your terminal type. You can get the path you need to add to your PATH by using which java.
  • After you have update your .bash_profile or .zshrc file you should reopen your terminal to let the changes take place. Then you can check your Java installation by entering java --version. If the version show up successfully then you have installed and setup Java correctly.
